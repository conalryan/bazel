## Ng-Bootstrap
1. Create worksapce
```bash
ng new --createApplication=false
# called ng-bootsrap
```

2. Add an app
```bash
ng g app my-app
```

3. Add a lib
```bash
ng g lib my-lib
```

4. Add ng-bootstrap
```bash
yarn add @ng-bootstrap/ng-bootstrap
# or
# npm install --save @ng-bootstrap/ng-bootstrap
```
